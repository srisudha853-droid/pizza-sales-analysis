# 🍕 Pizza Sales Analysis Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard created to analyze pizza sales performance, revenue trends, customer ordering behavior, and best/worst selling products.

The dashboard provides valuable business insights through interactive visualizations and KPI metrics.

---

## 🛠 Tools & Technologies Used

- Power BI
- Data Analytics
- Data Visualization
- DAX
- Excel

---

## 📊 Dashboard Features

- Revenue Analysis
- Order Trend Analysis
- Best & Worst Sellers
- Category-wise Sales Insights
- KPI Metrics
- Interactive Navigation Buttons
